# ESAB Rosetta - Claims Management System Demo

Advanced claims management platform demonstrating modernization from legacy ASPX .NET to modern React/Next.js architecture.

## Features

- **Premium UI Design**: Glassmorphism effects with ESAB brand colors (gold #FFD700 & black)
- **Login System**: Secure authentication interface
- **Dashboard**: Interactive claims overview with expandable sections
- **Claims Management**: 
  - Comprehensive 16-column claims table
  - Real-time status tracking
  - Advanced filtering and search
- **Ticket Details**: 7-tab interface including:
  - Ticket Details
  - Analysis
  - Corrective Action
  - Customer Notification
  - Assigned Tasks
  - Dates & History
  - General Comments
- **New Claim Creation**: 3 claim type options with comprehensive forms

## Technology Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Design**: Modern glassmorphism with CSS animations
- **Deployment**: Vercel
- **Version Control**: Git
